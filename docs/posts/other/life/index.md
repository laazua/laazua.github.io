##### 生活常识

- **年化利率**
::: code-group
```bash [定义]
# 年化利率,顾名思义,就是把任何一段时间内产生的利息或收益,
# 都“折算”成一整年的标准利率.它的核心作用是：
# 提供一个统一的标尺，让你能公平、直观地比较不同金融产品的成本(借钱)或收益(投资).

# 主要应用场景
# 1. 借贷场景(成本)
#     信用卡分期:
#         广告说"月费率0.5%",但年化利率可能高达12%以上,
#         因为每期都在还款,本金在减少,但手续费照旧收.
#     消费贷款/房贷/车贷:
#         银行通常直接给出年化利率(如LPR+基点),让你清楚贷款的成本.
#     网络借贷:
#         法律规定必须明示年化利率(APR),防止机构用"日息万五"等说法迷惑用户.

# 2. 投资理财场景(收益)
#     银行存款:
#         挂牌的都是年化利率.比如一年期定存利率1.75%,
#         存10000元,一年利息就是175元.
#     理财产品/基金:
#         常用"七日年化收益率"或"近一年年化收益率"来展示收益情况.
#         请注意:这只是基于过去数据的估算,不代表未来收益.
#     保险/国债:
#         也会提供预期的年化收益率作为参考.

# 必须分清的两对概念
# 1. 年化利率 (APR) vs 年化收益率 (APY)
#     年化利率:通常指名义利率,不考虑利息再投资或复利效应.常用于借贷.
#     年化收益率:考虑了复利(利滚利)效应后的实际收益率.常用于投资.
#     简单比喻:APR是"单利尺子",APY是"复利尺子".在复利情况下,APY > APR.

# 2. 单利 vs 复利
# 单利:利息只按最初本金计算,每期的利息不计入下一期的本金.
#     公式:
#         总利息 = 本金 × 利率 × 时间
#     复利:
#         利息会加入本金,一起计算下一期的利息(利滚利).
#     公式:
#         本息和 = 本金 × (1 + 利率)^期数
#     巨大差异:
#         长期来看,复利效应像滚雪球.例如,年化5%的复利,
#         10万块20年后约变成26.5万;而单利只有20万.

# 最重要的建议
#     任何借贷,看清"年化利率(APR)":
#         这是受法律监管、最能反映真实成本的数字.不要只看"月供”或"日息".
#     任何投资,警惕"预期年化收益率":
#         这只是预估,不是承诺.高收益必然伴随高风险.
#     善用工具:
#         网上有很多"年化利率计算器",在办理分期或贷款时,自己算一下,心里更有底.
#     牢记"72法则":
#         快速估算复利下本金翻倍需要的时间.
#         用 72 除以 年化收益率.例如,年化收益6%,大约72÷6=12年翻倍.
#     总结一下:
#         年化利率就是一把"财务标尺",把所有不同期限的金融产品的成本或收益,
#         统一换算成"一年的标准利率",方便你进行比较和决策.
#         它是你看清金融产品本质、避免被误导的最重要工具.
```

```bash [示例]
# 假设有两个人向你借钱：
#   小王 借1000元,3个月后还你1030元.
#   小李 借1000元,6个月后还你1050元.

# 问题：谁借钱的成本更高？

# 计算方法一：单利法（最常用、最直观）
#     核心公式：年化利率 = (利息 / 本金) × (一年时间 / 借款时间)
#     计算小王（借款3个月，利息30元）：
#     利息 = 1030 - 1000 = 30元
#     借款时间 = 3个月
#     一年时间 = 12个月
#     年化利率 = (30 / 1000) × (12 / 3) = 0.03 × 4 = 0.12，即 12%
#     计算小李（借款6个月，利息50元）：
#     利息 = 1050 - 1000 = 50元
#     借款时间 = 6个月
#     一年时间 = 12个月
#     年化利率 = (50 / 1000) × (12 / 6) = 0.05 × 2 = 0.10，即 10%
#     所以，用单利法计算，小王的年化成本（12%）确实高于小李（10%）。

# 计算方法二：复利法（更精确，但计算略复杂）
#     复利法假设这笔借款的利息模式可以按同样利率在一年内多次滚存。公式为：
#     本息和 = 本金 × (1 + 期利率) ^ 期数
#     我们需要反推期利率，再年化。
#     计算小王（3个月为一期，一年有4期）：
#     3个月后本息和：1030 = 1000 × (1 + r) ^ 1 （这里r是3个月的利率）
#     所以， 1 + r = 1030 / 1000 = 1.03
#     3个月的利率 r = 0.03
#     年化利率（复利） = (1 + 期利率)^年中期数 - 1 = (1 + 0.03)^4 - 1
#     计算：1.03^4 ≈ 1.1255
#     年化利率 ≈ 1.1255 - 1 = 0.1255，即约 12.55%
#     计算小李（6个月为一期，一年有2期）：
#     6个月后本息和：1050 = 1000 × (1 + r) ^ 1 （这里r是6个月的利率）
#     所以， 1 + r = 1050 / 1000 = 1.05
#     6个月的利率 r = 0.05
#     年化利率（复利） = (1 + 0.05)^2 - 1
#     计算：1.05^2 = 1.1025
#     年化利率 ≈ 1.1025 - 1 = 0.1025，即约 10.25%
```
