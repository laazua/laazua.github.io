##### [pyinfra](https://github.com/pyinfra-dev/pyinfra)

<pre>
pyinfra 一款配置管理工具,更加偏向编码的方式进行配置管理.
将 Python 代码转换为 shell 命令并在服务器上执行.
支持执行临时命令和编写声明式操作,可操作 SSH 服务器、本地机器及 Docker 容器.
运行迅捷,可从单台服务器扩展至数千台.
</pre>
